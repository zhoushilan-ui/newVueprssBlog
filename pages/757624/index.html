# textillatejs 基础知识点

# Textillate.js

Textillate.js是用于创建CSS3文本动画的简单插件。该插件结合了不同的库，以帮助将CSS3动画应用于任何文本。要使用它，您需要做的就是将textillate.js及其依赖项添加到您的项目中。textillate.js使用jquery. 要使用textillate.js，必须下载文字变色插件lettering.js (opens new window),animate.css (opens new window),jquery.

lettering.js的github/:https://github.com/davatron5000/Lettering.js

# CDN地址：

lettering.js的CDN:https://cdn.bootcdn.net/ajax/libs/lettering.js/0.7.0/jquery.lettering.js
animate.css的CDN:https://cdnjs.cloudflare.com/ajax/libs/animate.css/4.1.1/animate.min.css
jquery的CDN:https://cdn.bootcdn.net/ajax/libs/jquery/3.6.0/jquery.js

lettering.js的作用：对字母、单词、或者段落进行分割
lettering.js的思想：把单词分成一个一个的字母，然后对字母进行效果处理
